# Simple Calculator (Ruby)

This is a simple command-line calculator written in Ruby for beginners.

## Features
The program supports:
- Addition  
- Subtraction  
- Multiplication  
- Division (with zero division check)

## How to run

Make sure you have Ruby installed, then run:

ruby calculator.rb

## Example

Input:
First number: 10  
Second number: 5  
Operation: 1  

Output:
Result: 15
